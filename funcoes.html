Funções

Funções são blocos de código reutilizáveis em JavaScript.

Elas ajudam a organizar melhor o código e evitar repetições.

Uma função pode receber parâmetros e retornar valores.

Exemplo de função simples:

function saudacao() {
alert("Olá, Mundo!");
}

Função com Parâmetro:

function mostrarMensagem(nome) {
alert("Olá, " + nome + "!");
}

Função com Retorno:

function soma(a, b) {
return a + b;
}

Exemplo de uso: console.log(soma(3, 4));

Função Arrow:

const multiplica = (x, y) => x * y;
console.log(multiplica(2, 5)); // Resultado: 10
