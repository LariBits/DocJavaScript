Introdução ao JavaScript

JavaScript é a linguagem de programação da web.

Ele pode atualizar e alterar tanto HTML quanto CSS.

Por que estudar JavaScript?

JavaScript é uma das 3 linguagens que todos os desenvolvedores web devem aprender:

  • HTML para definir o conteúdo das páginas da web
  • CSS para especificar o layout das páginas da web
  • JavaScript para programar o comportamento de páginas web

JavaScript pode alterar o conteúdo HTML

Um dos muitos métodos JavaScript HTML é getElementById().

Exemplo:

document.getElementById("demo").innerHTML = "Hello JavaScript";
